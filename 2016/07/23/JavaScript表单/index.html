Javascript中的Form表单知识点总结

表单的基础知识

JavaScript中，表单对应的是HTMLFormElement类型的对象
独有的属性和方法：

  • acceptCharset | 服务器能够处理的字符集 | 等价于HTML中的accept-charset特征
  • action | 接收请求的URL | 等价于HTML中的action
  • elements | 表单中所有控件的集合（HTMLCollection）
  • enctype | 请求的编码类型 | 等价于HTML中的enctype
  • length | 表单中控件的数量
  • method | HTTP请求类型（post、get）| 等价于HTML中的metchod
  • name | 表单名称 | 等价于HTML中的name
  • reset() | 将所有表单域重置为默认值
  • submit() | 提交表单
  • target | 用于发送请求和接收响应的窗口名称 | 等价于HTML中的target

提交表单

调用submit()方法不会触发submit事件

重置表单

调用reset()方法会触发reset事件

表单字段

表单都有elements属性，是表单中所有表单元素（字段）的集合

表单字段的共有属性（除form外，可用JavaScript修改值）：

  • disable | 布尔值 | 表示当前字段是否被禁用
  • form | 指针 | 指向所属表单
  • name | 字段的名称
  • readOnly | 布尔值 | 表示是否只读
  • tabIndex | 表示当前字段的tab切换序号
  • type | 字段类型“checkbox”、“radio”等
  • value | 字段被提交给服务器的值

侦听submit事件，在第一次单击后禁用提交按钮

表单字段方法

focus() | 激活表单字段

blur() | 移除焦点

表单字段事件

  • blur | 失去焦点时触发
  • change | <input>、<textarea>失去焦点且value改变时触发，<select>选项改变时触发
  • focus | 获得焦点时触发

选择框脚本

HTMLSelectElement类型的属性和方法

  • add(newOption, relOption) | 向控件中插入新的option元素，在relOption之前
  • multiple | 布尔值 | 表示是否允许多选
  • options | 控件中所有的options元素的HTMLCollection
  • remove(index) | 移除给定位置的选项
  • selectedIndex | 基于0的选中项的索引，如果没有选中，则为-1。多选控件只保存第一项的索引
  • size | 选择框的可见行数 | 等价于HTML中的size特征

选择框的type属性为select-one和select-multiple

选择框的value为

  • 如果没有选中项，则为空
  • 如果有一个，HTML中已指定，则为value特征值
  • 如果有一个，HTML中未指定，则为该项文本
  • 如果有多个选中项，则选择第一个选中项的值

每个option元素都有一个HTMLOptionElement对象，该对象的属性：

  • index | 当前选项在options集合中的索引
  • label | 当前选项的标签
  • selected | 布尔值 | 表示是否被选中
  • text | 选项的文本
  • value | 选项的值

表单序列化

  • 表单字段的名称和值进行URL编码，使用&分割
  • 不发送禁用表单字段
  • 只发送勾选的复选框和单选按钮
  • 不发送type为reset和button的按钮
  • 多选框中每个选中的值单独一个条目
  • 单击提交按钮会发送提交按钮，否则不发送（包括type为image的input元素）
  • select元素的值，是选中的option的value特征的值，如果没有value特征，则是文本值

操作富文本

document.execCommand() 参数：执行命令名称，是否提供用户界面，执行命令的值

  • backcolor | 颜色字符串 | 文档背景颜色
  • bold | null | 文本转换为粗体
  • copy | null | 文本复制到剪贴板
  • createlink | URL字符串 | 文本转换成一个链接
  • cut | null | 文本剪切到剪贴板
  • delete | null | 删除选择文本
  • fontname | 字体名 | 修改字体
  • fontsize | 1~7 | 修改字体大小
  • formatblock | 要包围文本块的标签 | 增加HTML标签
  • indent | null | 缩进文本
  • inserthorizontalrule | null | 插入一个hr元素
  • insertimage | 图像URL | 插入一个图像
  • insertorderedlist | null | 插入一个ol元素
  • insertunorderedlist | null | 插入一个ul元素
  • insertparagraph | null | 插入一个p元素
  • italic | null | 转换成斜体
  • justifycenter | null | 居中对齐
  • justifyleft | null | 左对齐
  • outdent | null | 凸排文本（减少缩进）
  • paste | null | 粘贴剪贴板文本
  • removeformat | null | 移除块级格式（撤销formatblock）
  • selectall | null | 选择文档中所有文本
  • underline | null | 下划线
  • unlink | null | 移除链接
