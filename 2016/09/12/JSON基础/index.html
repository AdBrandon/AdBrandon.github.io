JSON基础

JSON(JavaScript Object Notation, JS 对象标记) 是一种轻量级的数据交换格式。JSON 语法是 JavaScript 语法的子集。

语法

结构化数据的格式

  • 简单值 | 字符串、数值、布尔值、null（不支持undefined）
  • 对象 | 无顺的键值对
  • 数组 | 有序的值得列表

简单值

对象

  1. 没有声明
  2. 末尾没有分号
  3. 属性必须加双引号

数组

解释和序列化

序列化

JSON.stringify()不包含任何空格字符或缩进，所有函数及原型成员会被忽略，值为undefined的属性也会跳过。

序列化选项

JSON.stringify()参数：JavaScript对象，过滤器，缩进选项
过滤器可以是一个数组，也可以是一个函数

1，如果过滤器参数是数组，则序列化结果只包含数组中列出的属性

2,如果过滤器参数是函数，传入函数有两个参数，属性名和属性值。根据属性名修改属性值

3，缩进选项，如果参数是一个数值，那它表示每个级别的缩进空格数。
最大为10，大于10的值都会自动转换成10

4，如果缩进选项为字符串，则这个字符串会被用作缩进字符

toJSON()方法

给对象定义toJSON()方法，返回其自身的JSON数据
在序列化对象时，首选会检查是否存在toJSON方法，如果有，就按该方法执行

解释器

JSON.parse()如果参数不是有效的JSON，该方法会抛出错误。

解析选项

JSON.parse()方法也可以接收另一个参数，该参数是一个函数，将在每个键值对上调用
